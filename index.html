User information form

You can see rules those I used when made this widget below the form

  • There are some rules I used when I made this widget
  • The main rule:
  • Validate while typing
  • Email rules:
  • An email shouldn't have any spaces
  • The symbol @ is required
  • These emails have already been taken: 'author@mail.com', 'foo@mail.com', 'tester@mail.com'
  • Password rules:
  • A password should have latin letters and numbers only
  • A password should have no less than one letter and one number
  • A password should have no less than five symbols
  • Phone rules:
  • A phone number should have an international format +X
  • A phone number should have only numbers or symbols +, - or brackets
  • A phone number should have not less than 11 numbers (another symbols are not counted)
  • A phone number should have not more than 18 numbers (another symbols are not counted)
  • Other rules:
  • All required fields should be validated
  • Checkbox 'I agree with all' should be confirmed
